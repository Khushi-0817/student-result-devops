\# Student Result Management System (DevOps Project)



\## Project Overview

This project is a Maven-based Student Result Management System developed as a DevOps-oriented application. The main objective of the project is to demonstrate build automation, dependency management, and version control using Maven and Git. The application calculates total marks, percentage, and grade for a student and generates a runnable JAR file.



---



\## Maven Build Lifecycle

The project follows the standard Maven build lifecycle:

\- validate

\- compile

\- test

\- package

\- install



Maven is used to automate compilation, testing, and packaging of the application into a deployable JAR file.



---



\## Git Workflow

Git is used for version control and source code management. The workflow includes:

\- Initializing a local Git repository

\- Committing source code and configuration files

\- Linking the local repository to GitHub

\- Pushing changes to a remote repository using secure authentication



This workflow ensures traceability, collaboration readiness, and CI/CD compatibility.



---



\## How to Build and Run the Project

Run the following commands from the Command Prompt:



Build the project:

mvn clean package

Run the application:

java -jar target/my-app-1.0-SNAPSHOT.jar
